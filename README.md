# Commenting Module

![Js Logo][1]  [![Build Status](https://travis-ci.org/anistark/commenting-react.svg?branch=master)](https://travis-ci.org/anistark/commenting-react)


## Pre-Required :

 - NodeJS > 6
 - NPM > 3


## Packages :

 - react
 - webpack
 - babel
 - muicss


## Setup

```
npm install
```


## Run :

```
npm run dev
```


### To run to dist chrome :

```
<path-to-chrome> src/client/index.html
```
